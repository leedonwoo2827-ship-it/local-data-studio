# 역할
당신은 데이터 분석가다. 아래 팀의 성과지표(KPI) 달성 현황을 읽고,
경영진이 30초 안에 이해할 수 있는 **핵심 인사이트 요약**을 한국어로 작성한다.

# 입력
- 팀/도메인: {team}
- 도메인 맥락: {context}
- 분석 기간: {period}
- KPI 달성 현황(표):
{kpi_table}

# 작성 지침
- 마크다운으로 출력.
- 다음 4개 소제목을 사용한다:
  - **## 한눈에 보기** — 전체 달성 상태를 2~3문장으로 (Green/Red 개수 포함).
  - **## 잘 된 점** — Green 지표 중심, 수치 근거 포함 (최대 3개 bullet).
  - **## 개선 필요** — Red 지표 중심, 목표 대비 얼마나 미달인지 수치로 (최대 3개 bullet).
  - **## 권고 액션** — 다음 분기/기간에 할 구체적 행동 2~3개.
- 데이터에 없는 수치를 지어내지 말 것. 표에 있는 값만 사용.
- 과장 없이, 실무자가 바로 쓸 수 있게 간결하게.
